# 別途必要な部品の用意(オプション含む)
以下の部品を用意します。

* 7mmオプティカルトラックボールモジュール１uタイプ(ADTB7M)

* トラックボールモジュール用レベル変換基板(以下、変換基板と呼びます。)

いずれも遊舎工房様の実店舗、[通販サイト](https://yushakobo.jp/shop/)等で購入可能です。

# 組立時の主な注意点
* メイン基板の組立時にロープロファイルピンソケット1x4をはんだ付けしていない場合、トラックボールモジュールの取付は出来ません。ご注意ください。

　(組立後のロープロファイルピンソケット1x4の取付けは、邪魔になるキースイッチを取り外した後にトッププレートの一部を削れば可能かと思いますが、きっと大変です。今度一回試してみます。）

* トラックボールモジュールの配線を変換基板にはんだ付けする際は、変換基板の向きにご注意ください。

* トラックボールモジュール用レベル変換基板にピンヘッダ1x4を取付ける際は、端子の位置にご注意ください。(NC端子にはピンヘッダは取付けません。)

# (左キーボードを組立済の場合)ProMicroカバー、OLED、ProMicro、M2x12mmスペーサー、キースイッチの取外し

組立とは逆の手順で、ProMicroカバー、OLED、ProMicro、M2x12mmスペーサー、キースイッチの順に取外していきます。

キースイッチの取外しには、はんだの吸取が必要です。はんだの吸取には専用の道具が必要です。(私はHAKKO SPPON 20Gを使用しています。）

道具を購入するか、遊舎工房様の工作室等を利用する等、道具がある環境を用意した状態で作業してください。

# 変換基板へのトラックボールモジュール配線のはんだ付け

画像のように、変換基板へトラックボールモジュール配線をはんだ付けします。

各部品の向きに注意して下さい。

# 変換基板へのピンヘッダ1x4のはんだ付け

画像のように、変換基板へピンヘッダ1x4を取付け、はんだ付けします。ピンヘッダの短い端子が変換基板側です。

変換基板のNC端子にはピンヘッダは取付けないので注意して下さい。

# M2x4mmスペーサーの取付け

M2x10mmのねじ2本にM2x4mmスペーサーをそれぞれ取付けます。スペーサーを回転させたときにねじが回らなければOKです。

上側のスペーサーは、ねじの頭を爪楊枝等の細い棒で抑えながら取付けると良いです。

下側のねじは、一度ボトムプレートを取り外した後に、基板側から精密ドライバーでねじの頭を抑えながら取付ると良いです。

# トラックボールモジュール用ボトムプレートの穴へ変換基板を通す

画像の様に、トラックボールモジュール用ボトムプレートの穴へ変換基板を通します。

# ピンヘッダ1x4をロープロファイルピンソケット1x4へ取付け

画像の様に、変換基板へはんだ付けしたピンヘッダ1x4をロープロファイルピンソケット1x4へ取付けます。

# ボトムプレートのM2x10mmねじへの取付け

画像の様に、ボトムプレートをM2x10mmねじへ取付けます。

# M2x10mmねじへのM2x6㎜スペーサーの取付け

画像の様に、Mx10mmねじ2本へM2x6mmスペーサーをそれぞれ取付けます。

# トラックボールモジュールの配線位置の調整

トラックボールモジュールの配線を、メイン基板のトッププレートやトラックボールモジュール用ボトムプレートの穴へ通し、

トラックボールモジュールをボトムプレートの上に置けるようにします。

# ProMicro、OLEDの取付け

取り外しておいたProMicroやOLEDをメイン基板へ取付けます。

# トッププレートへの保護テープ貼付け

トッププレートの保護シールをはがした後、トッププレートの丸穴の部分へ保護テープを貼付けます。

(保護テープを貼る目的は、トラックボールを操作する際に丸穴のエッジを触った際の怪我防止です。)

保護テープを貼るのではなく、丸穴部分をカッターナイフの背などで糸面取りしてもOKです。

糸面取りについては、「アクリル　糸面取り」等で検索すると情報が得られると思います。

# トッププレートをM2x6mmスペーサーへM2x4mmねじでねじ止め

画像の様に、トッププレートの丸穴にトラックボールモジュールを保護テープがずれないよう気を付けながら取付けた後に、

M2x6mmスペーサー2つにトッププレートをM2x4mmねじ2本でねじ止めします。

取付が完了したら、トラックボールモジュールの取付は完了です。
